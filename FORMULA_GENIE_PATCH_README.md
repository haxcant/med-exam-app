## 覆蓋方式

將此 zip 解壓後覆蓋到 MED-EXAM 專案根目錄，提交到 GitHub Pages 對應分支即可。

## 主要檔案

- `index.html`：方劑精靈區塊文案與圖片路徑。
- `styles.css`：方劑精靈圖示、命中特徵明細與主題相容樣式。
- `formula_genie_matcher.js`：候選方劑的命中特徵與權重明細。
- `assets/formula-genie.png`：本版方劑精靈圖示。
- `service-worker.js`：快取版本與圖示資源更新。

## 基本驗證

1. 開啟首頁，展開「方劑精靈」。
2. 確認標題區不再重複顯示多段近似文字。
3. 確認圖示顯示為新的神秘風格圖片。
4. 輸入範例或自行輸入症狀敘述，按「揣測方劑」。
5. 每個候選方劑下方應可展開「命中特徵與權重」。
