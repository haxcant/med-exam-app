# med-exam-app v0.1.5 按鈕命名與說明整理 Patch

請覆蓋下列檔案到你的 `med-exam-app` 根目錄：

- `index.html`
- `app.js`
- `firebase-ui.js`
- `service-worker.js`
- `CHANGELOG.md`

不要覆蓋你的 `firebase-init.js`。

更新重點：

- 「匯出完整記憶」改為「匯出本機完整記憶備份」。
- 「匯入學習檔案」改為「匯入本機備份／錯題檔」。
- 「上傳雲端備份」改為「上傳本機記憶到雲端」。
- 「下載雲端備份」改為「下載雲端記憶到本機」。
- 使用說明新增按鈕用途速查，協助使用者理解每個按鈕。
