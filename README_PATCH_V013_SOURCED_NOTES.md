# v0.1.13 sourced notes patch

本 patch 針對《金匱要略》題庫補充說明進行第二輪加強：

- 將原先偏「換句話說」的註解改成較有辨證意義的說明。
- 每題補充說明維持以下欄位：
  - 原文
  - ChatGPT白話註解
  - 病機
  - 治法
  - 方義
  - 關鍵字記憶點
  - 參考來源
  - 使用提醒
- 每題均標示來源：上傳 PDF 頁碼、中醫笈成《金匱要略方論》；若該方有衛福部中醫藥司基準方頁面，則在該題註解中加入對應官方來源。
