# v0.1.7

- 錯題本列印版不再顯示「最近錯答：未保存」。
- 若舊版錯題或匯入檔沒有保存最近錯答選項，列印版會直接省略「最近錯答」欄位，避免讓使用者誤以為自己沒有保存好。
- 列印版標題區新增小字說明：太久以前的錯題或舊匯入檔可能不顯示最近錯答選項。
- 匯入錯題檢視頁同樣省略缺少最近錯答摘要的欄位。
- Service Worker cache bump 至 `med-exam-app-v0.1.7`。

# v0.1.6

- 修正錯題本列印版每題都顯示「未作答」的問題。
- 答題記憶改為「每題聚合資料 + 最近一次作答摘要」，不保存完整作答歷史，避免本機與雲端備份容量膨脹。
- 錯題本列印版改顯示「最近錯答」；若舊資料或匯入檔沒有保存作答摘要，會標示「未保存」，不再誤導為未作答。
- 匯入錯題 JSON / 列印版時會盡量保留最近錯答摘要。
- Service Worker cache bump 至 `med-exam-app-v0.1.6`。

# v0.1.5

- 統一主要按鈕命名，明確區分「本機檔案備份」與「雲端同步」。
- 將「匯出完整記憶」改為「匯出本機完整記憶備份」。
- 將「匯入學習檔案」改為「匯入本機備份／錯題檔」。
- 將雲端同步按鈕改為「上傳本機記憶到雲端」與「下載雲端記憶到本機」。
- 使用說明新增按鈕用途速查，提醒錯題本 JSON / 列印版可交給 GPT、NotebookLM 等 AI 做筆記整理。
- Service Worker cache bump 至 `med-exam-app-v0.1.5`。

# v0.1.4

- 將「下載本輪錯題（列印版）」改為「下載全部錯題本（列印版）」。
- 強化錯題 JSON / 錯題列印 HTML 匯入相容性，支援較鬆散的 wrong items envelope，並用題目 id / 題幹 / 答案嘗試對應目前題庫。
- 說明區新增 Chrome 相容性建議。
- 雲端下載套用成功後自動刷新一次，並提示若數據未更新可手動刷新。
- 說明區提示可將錯題檔交給 GPT / NotebookLM 等 AI 工具整理筆記。
