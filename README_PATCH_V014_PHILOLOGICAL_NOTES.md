# v0.1.14 國文註釋型白話註解修補

本 patch 重寫 `med_questions.js` 內 199 題的補充說明，目標是把先前偏「換句話說」的註解改成較接近國文課文註釋與中醫方證學習筆記的格式。

每題格式調整為：

```text
原文：
詞句註釋：
白話翻譯：
辨證要點：
病機：
治法：
方義：
關鍵字記憶點：
參考來源：
使用提醒：
```

本版特別修正：

- 「蝕於上部則聲喝」等古文詞句會先解釋「蝕」「上部」「聲喝／聲嗄」意思，再說方證。
- 刪除大量「病位、寒熱、虛實」類空泛句。
- 對常見方劑補入方義、治法與記憶點，例如甘草瀉心湯、半夏厚朴湯、大承氣湯、小青龍湯、五苓散、小柴胡湯、小建中湯等。
- 校正一個明顯自動抽題缺失：JGYL-0059 由「建中湯」修為「大建中湯」，並補回原條文主要症狀。
- 清理部分題幹的自動抽取殘字，例如「懸病懸飲」「飲咳家」「13渴欲」等。

## 使用提醒

本 patch 仍定位為「學習輔助與背誦提示」，不是正式校注、醫囑或臨床診療建議。若要升級成考證級校注，仍需逐條對照底本、注家與教材版本。
